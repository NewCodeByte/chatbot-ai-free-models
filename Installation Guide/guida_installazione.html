Chatbot AI Free Models

Grazie per aver scelto Chatbot AI Free Models! Questo chatbot ti permette di integrare un assistente virtuale personalizzabile nel tuo sito web, utilizzando modelli AI disponibili gratuitamente tramite OpenRouter.ai.

Caratteristiche Principali

  • Facile Integrazione: Aggiungi il chatbot a qualsiasi sito web con una singola riga di JavaScript.
  • Pannello di Amministrazione Sicuro: Un'interfaccia completa per configurare aspetto, comportamento, base di conoscenza, API, ecc., protetta da login.
  • Setup Guidato: Creazione sicura dell'account amministratore al primo accesso.
  • Password Sicure: Le password admin vengono memorizzate utilizzando hashing moderno (non in chiaro).
  • Personalizzabile: Modifica colori, icone, titolo, messaggio di benvenuto.
  • Flessibile: Definisci la base di conoscenza e le regole di comportamento del bot.
  • Memoria Conversazionale: Il bot può ricordare i messaggi precedenti all'interno di una sessione.
  • Supporto Multilingua: L'interfaccia del pannello admin e di login/setup è traducibile.
  • OpenRouter.ai Ready: Preconfigurato per usare i modelli di OpenRouter.ai.

Installazione Rapida

  1. Carica i File

    Decomprimi l'archivio e carica l'intera cartella del chatbot (es. chatbot-x/) sul tuo server web in una posizione accessibile via URL.

  2. Permessi Cartelle e File

    Assicurati che il server web abbia i permessi di scrittura per:

    • La cartella principale del chatbot (per creare e scrivere sul file database db.db).
    • La sottocartella images/ (per l'upload di icone/loghi).
    • La sottocartella sounds/ (per l'upload del suono di notifica).

    Verifica che il file db.db, una volta creato, sia anch'esso scrivibile dal processo del server web.

  3. Primo Accesso e Setup Admin

    Apri il tuo browser e naviga alla pagina di login: http://tuosito.com/percorso_del_chatbot/login.php.

    Al primo accesso, il sistema rileverà che non esiste un amministratore e ti presenterà un modulo di registrazione.

    Inserisci un Nome Utente Admin e una Password sicura (e confermala) per creare il tuo account. Questo account verrà salvato nel database db.db con la password protetta da hash.

    Dopo aver creato l'account, verrai reindirizzato (o potrai accedere) alla normale pagina di login.

  4. Accesso al Pannello Admin

    Una volta creato l'account, accedi a http://tuosito.com/percorso_del_chatbot/login.php usando le credenziali che hai appena definito.

    Dopo il login, sarai reindirizzato al Pannello di Controllo (admin.php).

  5. Configurazione Iniziale

    Dentro il Pannello di Controllo:

    • Vai alla tab Impostazioni.
    • Inserisci la tua API Key di OpenRouter.ai.
    • Seleziona un Modello AI (molti modelli gratuiti sono disponibili).
    • Personalizza la Base di Conoscenza e le Regole di Comportamento. Consulta la Tab Guida nel pannello per template e suggerimenti.
    • Esplora le altre tab (Stile, Immagini, Suoni) per personalizzare ulteriormente.
    • Seleziona la Lingua Interfaccia desiderata.

  6. Integra nel Tuo Sito

    Copia la riga di script fornita nella Tab Guida del pannello admin. Dovrebbe essere simile a:

    <script src="/percorso_del_chatbot/embed.js" defer></script>

    Incolla questa riga nelle pagine del tuo sito web dove vuoi che appaia il chatbot, preferibilmente prima della chiusura del tag </body>. Assicurati che il percorso allo script embed.js sia corretto.

Struttura dei File (Principali)

  • admin.php: Il pannello di amministrazione.
  • chatbot.php: Gestisce la logica e l'interfaccia del chatbot.
  • embed.js: Script per l'integrazione nel sito.
  • style.css: Fogli di stile.
  • admin.js & chatbot.js: Script JavaScript.
  • login.php: Pagina di login e setup iniziale.
  • export.php: Esportazione messaggi.
  • db_init.php: Script di inizializzazione database.
  • db.db: Database SQLite (creato al primo avvio).
  • /images/: Cartella immagini.
  • /sounds/: Cartella suoni.
  • /languages/: Cartella traduzioni.
  • guida_installazione.html: Questo file di guida.

Sicurezza

  • Password Hashing: Le password sono ora memorizzate in modo sicuro tramite hashing.
  • Protezione Directory: Considera sempre di proteggere la directory del chatbot o specifici file come admin.php con metodi aggiuntivi (es. .htaccess su server Apache) se desideri un livello di sicurezza superiore, limitando l'accesso solo a determinati IP o richiedendo un'autenticazione HTTP aggiuntiva. L'accesso diretto al database è già protetto dal file .htaccess.
  • Permessi File: Imposta i permessi al minimo necessario per il funzionamento, garantendo la scrittura solo dove serve (cartella principale per db.db, images/, sounds/).

Risoluzione Problemi

Consulta la sezione "Risoluzione Problemi Comuni" nella Tab Guida del pannello di amministrazione.

  • Errore DB al primo accesso a login.php: Assicurati che la cartella del plugin sia scrivibile dal server web per permettere la creazione del file db.db.
  • Altri problemi: verifica i log di errore PHP del tuo server.
