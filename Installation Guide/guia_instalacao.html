Instalação Rápida

  1. Carregar os Arquivos

    Descompacte o arquivo e carregue a pasta inteira do chatbot (ex: chatbot-x/) para o seu servidor web em um local acessível via URL.

  2. Permissões de Pastas e Arquivos

    Certifique-se de que o servidor web tenha permissões de escrita para:

    • A pasta principal do chatbot (para criar e escrever no arquivo de banco de dados db.db).
    • A subpasta images/ (para o upload de ícones/logos).
    • A subpasta sounds/ (para o upload do som de notification).

    Verifique se o arquivo db.db, uma vez criado, também possui permissão de escrita pelo processo do servidor web.

  3. Primeiro Acesso e Configuração do Admin

    Abra seu navegador e navegue até a página de login: http://seusite.com/caminho_do_chatbot/login.php.

    No primeiro acesso, o sistema detectará que não existe um administrador e apresentará um formulário de registro.

    Insira um Nome de Usuário Admin e uma Senha segura (e confirme-a) para criar sua conta. Esta conta será salva no banco de dados db.db com a senha protegida por hash.

    Após criar a conta, você será redirecionado (ou poderá acessar) à página de login normal.

  4. Acesso ao Painel de Admin

    Uma vez criada a conta, acesse http://seusite.com/caminho_do_chatbot/login.php usando as credenciais que você acabou de definir.

    Após o login, você será redirecionado para o Painel de Controle (admin.php).

  5. Configuração Inicial

    Dentro do Painel de Controle:

    • Vá para a aba Configurações.
    • Insira sua Chave de API da OpenRouter.ai.
    • Selecione um Modelo de IA (muitos modelos gratuitos estão disponíveis).
    • Personalize a Base de Conhecimento e as Regras de Comportamento. Consulte a Aba Guia no painel para modelos e sugestões.
    • Explore as outras abas (Estilo, Imagens, Sons) para personalizar ainda mais.
    • Selecione o Idioma da Interface desejado.

  6. Integrar em Seu Site

    Copie a linha de script fornecida na Aba Guia do painel de admin. Deve ser semelhante a:

    <script src="/caminho_do_chatbot/embed.js" defer></script>

    Cole esta linha nas páginas do seu site onde você deseja que o chatbot apareça, preferencialmente antes da tag de fechamento </body>. Certifique-se de que o caminho para o script embed.js esteja correto.

Estrutura de Arquivos (Principais)

  • admin.php: O painel de administração.
  • chatbot.php: Gerencia a lógica e a interface do chatbot.
  • embed.js: Script para integração no site.
  • style.css: Folhas de estilo.
  • admin.js & chatbot.js: Arquivos JavaScript.
  • login.php: Página de login e configuração inicial.
  • export.php: Exportação de mensagens.
  • db_init.php: Script de inicialização do banco de dados.
  • db.db: Banco de dados SQLite (criado na primeira execução).
  • /images/: Pasta de imagens.
  • /sounds/: Pasta de sons.
  • /languages/: Pasta de traduções.
  • guia_instalacao.html: Este arquivo guia (nome sugerido).

Segurança

  • Hashing de Senhas: As senhas agora são armazenadas de forma segura usando hashing.
  • Proteção de Diretório: Sempre considere proteger arquivos específicos como admin.php com métodos adicionais (ex: .htaccess em servidores Apache) se desejar um nível de segurança mais alto, limitando o acesso apenas a IPs específicos ou exigindo autenticação HTTP adicional. O acesso direto ao banco de dados já deve estar protegido pelo arquivo .htaccess fornecido.
  • Permissões de Arquivo: Defina as permissões ao mínimo necessário para a operação, garantindo permissão de escrita apenas onde for necessário (pasta principal para db.db, images/, sounds/).

Solução de Problemas

Consulte a seção "Solução de Problemas Comuns" na Aba Guia do painel de administração.

  • Erro de BD no primeiro acesso a login.php: Certifique-se de que a pasta do plugin tenha permissão de escrita pelo servidor web para permitir a criação do arquivo db.db.
  • Outros problemas: verifique os logs de erro PHP do seu servidor.
